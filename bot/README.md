# 🤖 بوت تداول أسهم أمريكية مُرمَّزة على Binance (bStocks)

بوت تداول متعدّد الأسهم يتداول **bStocks** — أسهم أمريكية **مُرمَّزة (tokenized)**
مطروحة على **Binance Spot** مقابل USDT (مثل `NVDABUSDT` لِـ NVIDIA و`TSLABUSDT`
لِـ Tesla). **يحدّث استراتيجيته تلقائياً** ليبقى متوافقاً مع حركة السوق، ويعرض
أداءه في لوحة ويب مباشرة.

> ⚠️ **تنبيه مهم:** هذه أداة **تعليمية وليست نصيحة مالية**. بوتات التداول قد
> **تخسر مالًا**. ابدأ دائمًا بوضع المحاكاة، ثم التجريبي، ولا تخاطر إلا بما
> تتحمّل خسارته. أنت وحدك المسؤول عن أي تداول حقيقي.

> 🌍 **ملاحظة أهلية:** bStocks **غير متاحة للأشخاص الأمريكيين** ولا في بعض
> المناطق (تُصدرها BTech Holdings ضمن إطار ADGM). تأكّد أن حسابك ومنطقتك
> مؤهَّلان قبل الوضع التجريبي/الحقيقي.

---

## ما الفرق عن بوت الكريبتو؟

نفس البنية والمنهجية بالضبط، مع **مكافئات للأسهم** بدل البيانات الخاصة بالكريبتو:

| الجانب | بوت الكريبتو | هذا البوت (أسهم) |
|--------|--------------|-------------------|
| الأصول | أزواج USDT كريبتو | bStocks (أسهم مُرمَّزة) على Spot/USDT |
| اتجاه السوق (بوابة 4) | BTC فوق MA200 | **S&P 500 (SPY) فوق MA200** (يومي، من Stooq مجاناً) |
| المال الذكي (بوابة 5) | نسبة Long/Short لفيوتشرز Binance | **القوة النسبية (RS)** مقابل S&P 500 |
| المشتقات/الفروث (بوابة 6) | تمويل/OI لفيوتشرز | **مزاج السوق (VIX) + جلسة التداول الأمريكية** |
| مزاج السوق | مؤشّر الخوف والطمع للكريبتو | **مؤشّر VIX** (مؤشّر الخوف للأسهم) |
| رأس المال الافتراضي | 1000 | **500** (أحجام صفقات أصغر) |

الوحدات غير المرتبطة بالمنصّة (`strategy`, `optimizer`, `backtest`, `ml_model`,
`indicators`) **منقولة كما هي**.

---

## كيف «يتعلّم ويحدّث نفسه»؟

افتراضيًا `REALTIME_LEARNING=true`، فيعيد البوت التعلّم على مدار `LEARN_SECONDS`:

1. **ضبط ذاتي بالباك-تيست** (`optimizer.py`) — يجرّب عشرات تركيبات الاستراتيجية
   (المتوسّطان، حدّ RSI، وقف الخسارة، جني الأرباح) على آخر بيانات السوق ويختار
   **أفضل تركيبة تعمّمت out-of-sample** لكل سهم.
2. **نموذج تعلّم آلي (ML)** (`ml_model.py`) — مصنّف Gradient Boosting لكل سهم
   يتنبّأ باحتمال الصعود ويُستخدم **كتأكيد للدخول** بدقّة out-of-sample حقيقية.
3. **قراءة بيئة السوق** (`best_practices.py`) — أخبار الأسهم + **مؤشّر VIX**:
   يوقف الشراء وقت الأخبار السلبية القوية أو هلع VIX، ويصغّر الحجم وقت التقلّب.

> مكتبات ML اختيارية؛ بدونها يعمل البوت بالإشارة الفنية وحدها ويتخطّى فلتر ML.

---

## بوابات الدخول الستّ (كلها AND — لا يُشترى إلا بنجاحها جميعاً)

1. **إشارة فنية** — تقاطع متوسّطات (أو scalp).
2. **تأكيد ML** — النموذج يرجّح الصعود (وإلا يُتجاهَل إن لم يكن له أفضلية).
3. **بوابة الأخبار/VIX** — لا شراء عند أخبار سلبية قوية أو هلع في VIX.
4. **اتجاه السوق العام** — لا شراء طالما S&P 500 تحت متوسّطه MA200.
5. **القوة النسبية (RS)** — لا شراء إذا كان السهم متخلّفاً بوضوح عن S&P 500.
6. **جلسة السوق/المخاطرة** — تجنّب الشراء وقت هلع VIX، واختيارياً خارج ساعات
   التداول الأمريكية (`TRADE_RTH_ONLY`).

## إدارة المخاطر والخروج

حجم ثابت لكل صفقة، حدّ أقصى للصفقات المفتوحة، وقف خسارة، جني أرباح، وقف متحرّك
(trailing)، وقف ATR متكيّف مع تذبذب السهم، وحدّ خسارة يومي (kill-switch).

---

## أفضل ممارسات الأمان (اقرأها أولًا)

1. **محاكاة → تجريبي → حقيقي**؛ لا تقفز للأموال الحقيقية.
2. مفتاح API بصلاحية **«تداول» فقط**، مع **تعطيل السحب (Withdrawals)** و**تقييد IP**.
3. لا تضع المفتاح في الكود — يُقرأ من `bot/.env` المتجاهَل من Git.
4. خاطر بمبلغ صغير لكل صفقة (`QUOTE_PER_TRADE`)، وفعّل `DAILY_LOSS_LIMIT`.
5. الوضع الحقيقي **مقفول** بمتغيّر تأكيد صريح `CONFIRM_LIVE=I_UNDERSTAND_THE_RISK`.

## التشغيل السريع

```bash
pip install -r bot/requirements.txt
cp bot/config.example.env bot/.env      # ثم عدّله
python bot/trading_bot.py               # محاكاة (آمن)
python bot/trading_bot.py --once        # دورة واحدة فقط
```

### الوضع التجريبي (أموال وهمية)
1. أنشئ مفتاح API من <https://testnet.binance.vision>.
2. في `bot/.env`: ضع المفتاح والسر و`BOT_MODE=testnet`.

### الوضع الحقيقي (أموال حقيقية — بحذر شديد) 🔴
1. مفتاح API حقيقي: **Spot Trading فقط، السحب معطّل، IP مقيّد**.
2. في `bot/.env`:
   ```
   BOT_MODE=live
   CONFIRM_LIVE=I_UNDERSTAND_THE_RISK
   BINANCE_API_KEY=...
   BINANCE_API_SECRET=...
   ```

### اللوحة المباشرة
عند ضبط `PORT` (أو على Railway تلقائياً) افتح الرابط لرؤية الصفقات المفتوحة
والسجلّ المباشر، مع **زر بيع يدوي** لكل صفقة (محمي اختيارياً بـ `MONITOR_TOKEN`).

## مصادر البيانات (كلها مجانية بلا اشتراك)

- **Binance Spot API** (عام) — أسعار وشموع bStocks وتنفيذ الأوامر.
- **Stooq / Yahoo Finance** — سلسلة S&P 500 (SPY) و VIX اليومية (بلا مفتاح، مع
  تبديل تلقائي بين المصدرين إن حُجب أحدهما).
- **Finnhub** (اختياري، مجاني 60 طلب/دقيقة) — يفعّل تلقائياً عند ضبط
  `FINNHUB_API_KEY`: توصيات المحلّلين + تقويم الأرباح + أخبار الشركة للسهم الأصلي
  (NVDABUSDT → NVDA). بدونه يعمل البوت كما هو.

> إن تعذّر الوصول لأي مصدر، يتدهور البوت بأمان إلى «وضع محايد» ولا يتعطّل.
