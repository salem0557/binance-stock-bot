# 🤖 بوت استثمار قصير الأمد في الأسهم الأمريكية المرمَّزة (bStocks)

بوت **استثماري** (أفق ~3 أشهر) على **Binance bStocks** — أسهم أمريكية مُرمَّزة
(NVIDIA, Tesla, AMD…) على Spot مقابل USDT. **ليس مضارباً لحظياً**: يختار أقوى
الشركات ويحملها أسابيع/أشهر لينمّي رأس المال بهدوء، ويعرض كل شيء في لوحة ويب.

> ⚠️ **تنبيه:** أداة **تعليمية وليست نصيحة مالية**. قد تخسر مالاً. ابدأ بالمحاكاة،
> ولا تخاطر إلا بما تتحمّل خسارته.

> 🌍 **أهلية:** bStocks غير متاحة للأشخاص الأمريكيين وبعض المناطق، وBinance.com
> يحجب الخوادم الأمريكية — انشر من منطقة غير أمريكية بحساب مؤهَّل.

---

## المنهجية (استثمار بالعوامل، لا مضاربة)

يركب أقوى عاملين مثبتين على أفق 1–6 أشهر: **الزخم (Momentum)** و**الاتجاه
(Trend)**، مع ميل نحو **الجودة** (تقييم المحلّلين، نمو الأرباح) ومكافأة على
**التوزيعات (Dividends)**.

**الفكرة المحورية:** الرموز المرمَّزة عمرها أيام، لكن البوت يبني القرار من **تاريخ
السهم الأصلي الطويل** (NVDABUSDT → NVDA، سنوات من البيانات عبر Yahoo/Stooq)
وأساسياته (Finnhub)، ثم ينفّذ على الرمز المرمَّز الذي يتبع السهم 1:1.

### درجة التقييم (لكل سهم)
```
score = زخم(3 و6 أشهر)  +  اتجاه صاعد (فوق MA200/MA50)
      + قوة نسبية مقابل S&P500  +  ميل المحلّلين  +  مكافأة التوزيعات  +  نمو الأرباح
```
يحتفظ بأفضل **TOP_N** أسهم تحقّق درجة موجبة **واتجاهاً صاعداً**.

### بوابات الشراء
1. السهم ضمن أفضل المختارين بدرجة موجبة.
2. اتجاه صاعد للسهم (فوق متوسّط 200 يوم).
3. السوق العام صاعد (S&P 500 فوق MA200).
4. لا هلع في VIX (واختيارياً داخل ساعات السوق الأمريكية).
5. لا أخبار سلبية قوية.

### الخروج (تداول يومي — جني أرباح صغيرة وإقفال يومي)
- **جني ربح** صغير لكل صفقة (افتراضي +2%).
- **وقف خسارة** ضيّق (افتراضي ‎-3%).
- **كسر الاتجاه**: خروج عند هبوط السهم الأصلي تحت متوسّط 200 يوم.
- **إقفال قبل إغلاق السوق** (`CLOSE_AT_SESSION_END`): لا مبيت — يبيع كل شيء قبل
  16:00 ET ويكمل غداً.

### هدف الربح اليومي 🎯
`DAILY_PROFIT_TARGET` (USDT): عند بلوغ ربح اليوم المحقّق هذا المبلغ، **يقفل كل
الصفقات ويتوقّف حتى الغد**. هكذا «يثبّت ربحاً يومياً ثم يتوقف». 0 = معطّل.

> يتداول فقط أثناء جلسة السوق الأمريكية (`TRADE_RTH_ONLY=true`)، ويعيد اختيار
> الأسهم كل `OPTIMIZE_HOURS` (افتراضي 4 ساعات).
>
> ⚠️ **بصراحة:** لا يمكن **ضمان** ربح كل يوم — السوق قد يكون أحمر. ما يضمنه البوت
> هو **تثبيت الربح فور بلوغ الهدف ثم التوقّف**، وعدم تحويل يوم خاسر إلى ربح بالقوّة
> (ذلك يعني الاحتفاظ بالخاسرين = خسارة أكبر).

## الأوضاع الثلاثة
`dryrun` (محاكاة، الافتراضي) ← `testnet` ← `live` (حقيقي، يتطلّب
`CONFIRM_LIVE=I_UNDERSTAND_THE_RISK`).

## التشغيل السريع
```bash
pip install -r bot/requirements.txt
cp bot/config.example.env bot/.env      # ثم عدّله
python bot/trading_bot.py               # محاكاة
python bot/trading_bot.py --once        # دورة واحدة
```

### مفتاح Finnhub (مجاني ومُوصى به بشدّة)
سجّل في <https://finnhub.io> وضع `FINNHUB_API_KEY=...` — يضيف تقييم المحلّلين
ونمو الأرباح والتوزيعات للسهم الأصلي. بدونه يرتّب على الزخم/الاتجاه فقط.

### اللوحة المباشرة
اضبط `PORT` (أو على Railway تلقائياً) لرؤية المحفظة والتقييم الاستثماري والسجلّ
المباشر، مع زر بيع يدوي محمي بـ `MONITOR_TOKEN`.

## مصادر البيانات (كلها مجانية)
- **Binance Spot API** — أسعار وتنفيذ أوامر bStocks.
- **Yahoo Finance / Stooq** — تاريخ الأسهم الأصلية اليومي + S&P 500 + VIX (تبديل
  تلقائي إن حُجب أحدهما).
- **Finnhub** (اختياري، مجاني) — تقييم المحلّلين + التوزيعات + نمو الأرباح.

> أي مصدر يسقط ⇒ البوت يصير محايداً ولا يتعطّل.

## ملاحظات صريحة عن الحدود
- لا يمكن باك-تيست هذه الرموز (عمرها أيام)؛ المنهجية تعتمد على عوامل الاستثمار
  المعروفة لا على تحسين مُفرَط.
- الزخم/الجودة عوامل ذات سند بحثي على آجال أشهر، لكنها **ليست ضماناً** للربح.
